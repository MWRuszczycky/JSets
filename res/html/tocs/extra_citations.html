Extra articles

Extra articles can be specified here. First, click the Add an extra article button below. Next, use the drop-down to specify one of the following identifiers to find the abstract and enter its value in the adjacent text box:

  • PMID (PubMed ID). This is usually an eight-digit number that can be found at the PubMed website. For example, only enter the red text of:
    PMID: 28874588

  • DOI (the digital object identifier). For example, only enter the red text of:
    https://doi.org/10.1073/pnas.1710496114

  • LINK (a direct link to the abstract). For example, only enter the red text of:
    https://www.pnas.org/content/114/39/10408

Make sure the article can be found! Click the Check link button and make sure the correct abstract opens in a new browser tab. If it does not, then make sure your identifier correctly matches one of the above patterns or try another identifier.

To remove a selected article: Click the Remove button.
