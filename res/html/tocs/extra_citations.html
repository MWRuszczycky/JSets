Extra articles

@{citations}

Additional articles can be specified here. First, click the Add an extra article button below. Next, use the drop-down to specify a locator (i.e., DOI, WEB address or PubMed ID) to find the abstract and enter its value in the adjacent text box.

Click the triangle for examples how this article could be located. Click here for the example abstract at PubMed.

  • DOI (the digital object identifier). Any of the following would work for the example,
    https://doi.org/10.1073/pnas.1710496114
    doi.org/10.1073/pnas.1710496114
    10.1073/pnas.1710496114

  • WEB (the web address of the abstract). Any of the following would work for the example,
    https://www.pnas.org/content/114/39/10408
    www.pnas.org/content/114/39/10408

  • PMID (PubMed ID). This is usually an eight-digit number that can be found at the PubMed website. So, this would work for the example,
    28874588

Check to see if the article can be found. Click the Check link button and make sure the correct abstract opens in a new browser tab. If it does not, then make sure your locator is correct or try another.
However, even if your locator does not work, it will still be recorded.

To remove a selected article: Click the Remove button.
